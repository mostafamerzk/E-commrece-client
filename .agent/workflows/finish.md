# Workflow: Task Completion
# Trigger: /done

1. Ask the user for a brief summary of what was accomplished.
2. Update `docs/memory.md`:
    - Mark the current Task ID as `[x] Done`.
    - Add the current date and summary to "Recently Completed Tasks".
    - Log major changes in the "Update Log" section.
3. Confirm: "Memory updated. Great progress today!"
